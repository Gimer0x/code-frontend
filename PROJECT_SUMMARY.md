# Code.DappDojo.com - Executive Summary

## Project Overview

**Code.DappDojo.com** is a production-ready, interactive Web3 development learning platform that transforms how developers learn Solidity and smart contract development. Unlike traditional video-based courses, DappDojo emphasizes hands-on coding with real-time compilation, automated testing, and AI-powered assistance.

**Website:** https://code.dappdojo.com  
**Status:** Live Production Platform  
**Mission:** "Learn by doing, stop tutorials, it's time to get your hands dirty!"

## Technical Architecture

**Frontend:** Next.js 15, React 18, TypeScript, Tailwind CSS  
**Authentication:** NextAuth.js with Google OAuth and email/password  
**Compilation:** Solidity compiler (Solc 0.8.x) via Foundry service on Fly.io  
**Testing:** Automated Foundry test framework integration  
**Backend:** RESTful API with JWT authentication  
**Hosting:** Vercel (frontend), scalable microservices architecture
